# Brawlball Ranked Bot

## Installation
- requires Node.js >=14
- suggested: Visual Studio Code with ESLint-extension
- clone repository
- `npm install`
- insert `src/token.ts` (template: `src/token-template.ts`)
- `npm run build`
- `npm start`
